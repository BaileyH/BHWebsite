Each of the following 15 puzzles displays a grid and a small program that uses if-statements and robot commands. Trace the code for each puzzle and move the robot (using the arrow keys or WSAD) to where it would end up if the code were to be executed. Both the location and the direction need to be correct.

If the robot is directed to move onto a grey square, it “crashes,” and the program ends. Use the radio buttons to indicate whether or not the robot crashes.

There are a few patterns to the ways if-statements are typically used:
● Basic If-statements
● Sequential If-statements
● Basic If-else statements
● Nested If and if-else statements.
● Combinations of all of the above

Each set of three puzzles presents an example of one of these common patterns.
